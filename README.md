# URL Shortener 🔗

A URL shortener built with Node.js, Express, and MongoDB that generates short links and redirects users to original URLs.

---

## 🚀 Features

- Generate short URLs
- Redirect to original URLs
- Store URLs in MongoDB
- REST API based backend

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Nanoid
